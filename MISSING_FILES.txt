ВАРИАНТ 1 - Вручную:
───────────────────────────────────────────────────────────
1. Откройте папку вашего Cursor проекта
2. Найдите папки css/ и js/
3. Скопируйте файлы:
   - css/styles.css → green-valley-fixed/css/
   - css/mobile.css → green-valley-fixed/css/
   - js/app.js → green-valley-fixed/js/
   - js/mobile.js → green-valley-fixed/js/


ВАРИАНТ 2 - Через командную строку:
───────────────────────────────────────────────────────────
Windows (cmd):
copy "C:\путь\к\cursor\green-valley\css\*.css" ".\css\"
copy "C:\путь\к\cursor\green-valley\js\*.js" ".\js\"

Windows (PowerShell):
Copy-Item "C:\путь\к\cursor\green-valley\css\*.css" -Destination ".\css\"
Copy-Item "C:\путь\к\cursor\green-valley\js\*.js" -Destination ".\js\"

Linux/Mac:
cp /путь/к/cursor/green-valley/css/*.css ./css/
cp /путь/к/cursor/green-valley/js/*.js ./js/


ВАРИАНТ 3 - Если у вас есть файлы:
───────────────────────────────────────────────────────────
Просто перетащите (drag & drop) файлы в соответствующие папки


═══════════════════════════════════════════════════════════
  🚀 ПОСЛЕ КОПИРОВАНИЯ ФАЙЛОВ:
═══════════════════════════════════════════════════════════

1. Убедитесь что все 4 файла скопированы
2. Проверьте структуру:
   green-valley-fixed/
   ├── css/
   │   ├── styles.css  ✅
   │   └── mobile.css  ✅
   ├── js/
   │   ├── app.js      ✅
   │   └── mobile.js   ✅
   └── ... остальные файлы

3. Запустите сервер:
   node server.js

4. Откройте в браузере:
   http://localhost:3000

5. Подключитесь с телефона:
   http://[ВАШ_IP]:3000
   (IP будет показан в консоли)
